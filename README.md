# Personal Portfolio v1

This is a personal website I built to showcase my journey and progress as a Software Engineering student.

It serves as a central place to highlight the projects I've worked on, the technical skills I'm developing, and what I am currently learning as I work toward my degree.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Properties, Grid, Flexbox)
- **Typography:** Instrument Serif, Manrope, DM Mono
- **Icons:** FontAwesome

## 🚀 Key Features
- Responsive design with CSS Grid
- Dynamic scroll animations using Intersection Observer API
